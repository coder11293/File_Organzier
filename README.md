# File Organizer System

## Overview

The File Organizer System is a simple command-line application developed in Python that helps users manage files in the current working directory. It provides basic file operations such as creating, viewing, reading, editing, and deleting files through a user-friendly menu-driven interface.

## Features

* Create new files
* View all files in the current directory
* Read file contents
* Edit existing files
* Delete files
* Exception handling for common file errors
* Simple command-line interface

## Technologies Used

* Python
* OS Module
* File Handling
* Exception Handling

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/file-organizer-system.git
```

2. Navigate to the project folder:

```bash
cd file-organizer-system
```

3. Run the program:

```bash
python file_organizer.py
```

## Menu Options

1. Create a new file
2. View all files
3. Delete a file
4. Read a file
5. Edit a file
6. Exit
